# 贡献者公约（Contributor Covenant 行为准则）

## 我们的承诺

身为社区成员、贡献者和领导者，我们承诺使社区参与者不受骚扰，无论其
年龄、体型、可见或不可见的缺陷、族裔、性征、性别认同和表达、经验水
平、教育程度、社会与经济地位、国籍、相貌、种姓、肤色、宗教信仰、性
取向或性身份如何。

我们承诺以助益开放、友善、多样、包容、健康社区的方式行事和互动。

## 我们的标准

有助于为我们的社区创造积极环境的行为示例包括但不限于：

- 表现出对他人的同理心和善意
- 尊重不同的主张、观点和感受
- 提供和大方接受建设性意见
- 承担责任并向受我们错误影响的人道歉，并从经验中吸取教训
- 关注社区整体利益，而非仅是个人利益

不可接受的行为示例包括：

- 使用情色化的语言或图像，及性引诱或挑逗
- 嘲讽、侮辱或贬损性的评论，以及人身或政治上的攻击
- 公开或私下的骚扰行为
- 未经他人明确许可，公布其私人信息，例如住址或电子邮箱
- 其他有理由认定为违反职业操守的行为

## 落实之责

社区领导者有责任阐释和落实我们认可的行为标准，并妥善公正地处理任何
他们认定为不当、威胁、冒犯或有害的行为。

社区领导者有权且有责任删除、编辑或拒绝评论、提交、代码、维基编辑、
议题和其他不符合本行为准则的贡献，并在适当时机就管理决定背后的原因
进行说明。

## 适用范围

本行为准则适用于所有社区场合，也适用于在公共场合代表社区时的个人。

代表我们社区的情形包括但不限于：使用官方电子邮件地址、通过官方社交
媒体账户发帖，或在线上或线下活动中担任指定代表。

## 报告

辱骂、骚扰或其他不可接受的行为可通过 GitHub 仓库提供的私下渠道
（[Security Advisories](https://github.com/szgenle/zhmm-tauri/security/advisories/new)
或仓库公开联系方式）报告给项目维护者。所有投诉都将被迅速且公平地审查
和调查。

所有社区领导者都有义务尊重报告者的隐私和安全。

## 处置准则

社区领导者将遵循下述社区处置影响指南来明确他们认定违反本行为准则的行
为应受何种处置：

### 1. 更正

**社区影响**：使用不恰当的语言或其他在社区中被认定为不专业或不受欢迎
的行为。

**处置后果**：由社区领导者发出非公开的书面警告，明确说明违规行为的本
质，并解释为何处置。或要求公开道歉。

### 2. 警告

**社区影响**：单一事件或一系列违规行为构成的违规。

**处置后果**：警告并对连续行为施以处置。在指定时间内，不得与相关人员
互动，包括主动与行为准则执行者互动。这包括避免在社区场合及外部渠道
（如社交媒体）中的互动。违反这些条款可能会导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区准则，包括持续的不当行为。

**处置后果**：在指定时间内，暂时禁止与社区进行任何形式的互动或公开
交流。在此期间，不得与相关人员主动互动，包括主动与行为准则执行者互
动。违反这些条款可能会导致永久封禁。

### 4. 永久封禁

**社区影响**：行为模式违反社区准则，包括持续的不当行为、骚扰个人，或
攻击或贬低某个群体。

**处置后果**：永久禁止在社区内进行任何形式的公开互动。
